# AI 파이트 리그 — Slice 5 (AI 부트캠프 커리어)

성격 한 줄로 AI를 만들고, 우마무스메식 커리어로 키워, 스타일·스킬로 싸운다. 기획서: `../docs/GAME_DESIGN.md` (v0.8)

## 현재 기능

- 봇 생성 60초 플로우 (성격→4축·트리거·필살기·외형 자동 생성)
- 손그림 스타일 렌더러 (보일링 라인, 61,440+ 파츠 조합, 개체별 파스텔 배색)
- 결정론적 자동전투 + 리치 스타디움 연출 (관중·네온·파티클·필살기 컷인)
- 방치 훈련(30분당 1점, 8시간 캡) + 훈련 종목 선택 + 스탯 캡 40
- 로컬 리그: 두부→챔피언 5티어 승강제, 출전권 5장/일, NPC 30 (티어 난이도 보정)
- 기록 탭: 최근 20경기 + 결정론 리플레이 (시드+스냅샷 재실행)
- **주인 닮아가기**: 스킵·출석·수확 습관이 봇 성향에 서서히 반영 (축당 ±15, 하루 ±2 캡) — 봇이 말풍선으로 언급
- **온라인 (Slice 3)**: 진짜 유저 봇과 비동기 대전, 글로벌 랭킹 TOP 50, 관전 공유 링크(`?watch=<id>`) + 유입 CTA. 오프라인이면 로컬 NPC로 자동 폴백.
- **반려동물 (Slice 4)**: 밥/놀기/재우기 돌봄 + 폰 신호 기분 → "오늘 컨디션"이 전투에 반영.
- **AI 부트캠프 커리어 (Slice 5)**: 우마무스메식 12턴 육성을 AI로 리스킨. 트레이닝(발열=오버피팅 리스크)·동기·스토리 이벤트·평가전(벤치마크) → 배포(등급 S~D)·버전업. 전투는 **파이팅 스타일(돌격/기교/방어/일격) + 커리어에서 얻은 스킬** 발동. `career.js`·`skills.js`·`events.js`.
- 인터뷰 카드 (손글씨 폰트) PNG 저장

## 배포

**라이브: https://nick1148.github.io/botleague/**

정적 사이트 (빌드 불필요). GitHub Pages(`master` 브랜치 루트)로 배포 — `git push`만 하면 자동 재배포된다.

```
git push          # master → Pages 자동 빌드
```

백엔드: Supabase `couple-invest` 프로젝트의 `botleague` 격리 스키마 (`db/001_botleague_init.sql`).
접속 정보는 `src/config.js` (anon 키는 공개 전제 — 보안 경계는 RLS+RPC, §14.5).
CORS 확인됨: Supabase가 `https://nick1148.github.io` origin 허용.

## 실행

```
npm start
```

→ http://localhost:8123 (ES 모듈이라 file:// 직접 열기 불가)

- `/` — 게임 본편: 봇 생성 → 데뷔전 관전 → 인터뷰 카드 저장
- `/botlab.html` — 봇 실험실: 파츠·표정 갤러리 (귀여움 관문 검증용)

## 테스트

```
npm test
```

로직 모듈만 자동 테스트 (RNG·성격 파싱·파츠·전투 시뮬·대사). 렌더러·연출은 botlab.html과 본편에서 수동 관문으로 검증한다.

## 구조

`src/` 아래 모듈별 1책임. 결정론 규칙: 전투·대사·외형은 전부 시드 기반 —
같은 입력이면 언제나 같은 결과 (Slice 3 리플레이 공유·서버 검증의 전제).

- 상성 순환 (몬테카를로 실측): 교활>저돌 0.695 / 신중>교활 0.535 / 저돌>신중 0.738
